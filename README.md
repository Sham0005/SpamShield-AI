# 🛡️ SpamShield AI
An AI-powered Spam Message Detection System built using Machine Learning and Natural Language Processing (NLP). The application classifies text messages as **Spam** or **Ham (Not Spam)** in real time through a simple and interactive web interface.
---

## 🚀 Features
- 📩 Real-time Spam Message Detection
- 🤖 Machine Learning-based Classification
- 📝 TF-IDF Text Vectorization
- ⚡ Multinomial Naive Bayes Algorithm
- 🌐 Responsive Web Interface
- 📊 Confidence-based Prediction
- 💻 Easy-to-use Interface
---

## ▶️ How to Run
1. Clone the repository
```bash
git clone https://github.com/Sham0005/SpamShield-AI.git
```

2. Open the project folder
```bash
cd SpamShield-AI
```

3. Install the required packages
```bash
pip install -r requirements.txt
```

4. Run the application
```bash
python app.py
```

5. Open your browser and visit
```
http://127.0.0.1:5000
```
